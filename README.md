# 🎙️ YouTube Live Parser — умный парсер эфиров

Автоматическое выделение гостей по фразе "с нами на связи" + объединение в смысловые блоки.

## 🚀 Возможности

- 🕐 Поддержка 3 форматов временных меток: `2:27:42`, `14:07`, `2 минуты 15 секунд`
- 🔍 Автоматическое определение гостей по фразе "с нами на связи"
- 📦 Объединение монологов одного гостя в смысловые блоки
- 📊 Статистика: количество блоков, символов, слов
- 🎨 Визуальное разделение ведущих и экспертов
- 💻 Работает локально, без сервера, без API

## 📥 Быстрый старт

1. Скачайте `smart_parser_final.html` (или `index.html`)
2. Откройте файл в любом браузере
3. Загрузите файл расшифровки (.txt)
4. Нажмите кнопку "Разобрать на блоки"
