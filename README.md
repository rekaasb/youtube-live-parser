# 🎙️ YouTube Live Parser — анализ эфиров

Инструменты для работы с расшифровками YouTube-эфиров: разбивка на смысловые блоки и классификация речи с помощью ИИ.

## 📦 Инструменты

### 1. Парсер (`index.html`)
Разбивает расшифровку на смысловые блоки по гостям.
- Поддерживает 3 формата временных меток
- Автоматически выделяет гостей по фразе "с нами на связи"
- Объединяет монологи в цельные блоки
- Работает локально, без интернета

### 2. Анализатор (`classifier_blocks.html`)
Анализирует каждый блок через DeepSeek API и определяет тип речи.
- **ФАКТ** — проверяемое утверждение (цифры, даты, события)
- **ОЦЕНКА** — мнение, интерпретация, прогноз
- **ШУМ** — связки, междометия, повторы

**Результат:** для каждого гостя — % фактов, % оценок, коэффициент доверия.

## 🚀 Как использовать

### Для парсера
1. Скачайте `index.html`
2. Откройте в браузере
3. Загрузите файл расшифровки (.txt)
4. Нажмите "Разобрать на блоки"

### Для анализатора
1. Получите API-ключ на [platform.deepseek.com](https://platform.deepseek.com) (бесплатный кредит $5)
2. Скачайте `classifier_blocks.html`
3. Откройте в браузере
4. Вставьте API-ключ
5. Загрузите расшифровку
6. Нажмите "Анализировать"
